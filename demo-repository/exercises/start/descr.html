Un exercice simple pour savoir si on compris le fonctionnement.

Consigne

Vous devez ré-implémenter une fonction de base sans vous servir des fonctions déjà présentes.

  1. Ecrire la fonction rev de type 'a list -> 'a list, qui doit renvoyer la liste inversée
  2. Ecrire la fonction append de type 'a list -> 'a list -> 'a list
